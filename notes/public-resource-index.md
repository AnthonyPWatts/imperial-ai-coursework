# Public Resource Index

Curated public resources to support preparation for the Imperial College Business School **Professional Certificate in Machine Learning and Artificial Intelligence**.

The intent is to collect legitimate, public, course-adjacent material that can help with:

- linear algebra refresh
- Python / NumPy / pandas practice
- ML concept reinforcement
- capstone preparation
- repo/notebook organisation

---

## 9. Practical Rule

Use public material in this order:

```text
Official course material
Official Imperial-adjacent maths resources
My own notebooks
Prior-student repos for structure only
Solution repos only after serious independent attempts
```

If a resource shows exact quiz answers, graded assignment solutions or capstone submissions, treat it as a debugging reference only — not as learning source material.
